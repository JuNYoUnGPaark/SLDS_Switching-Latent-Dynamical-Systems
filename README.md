<h1 align="center">SLDS (Switching-Latent-Dynamical-Systems)</h1>

<p align="center">
  <em>Switching-Latent-Dynamical-Systems for Human Activity Recognition</em>
</p>



<p align="center">
  <b>Main Idea</b>: 기존의 복잡한 Attractor 기반 흐름 제어를 제거하고,<br/>
  다중 동역학 모드(Mode) 간의 전환에만 집중하여 HAR을 수행하는 모델.
</p>

<p align="center">
  "전통적인 시계열 제어 이론(SLDS)을 DL의 Mixture of Experts(MoE) 메커니즘과 결합하여,<br/>
  인간 행동의 미세구조(Sub-structure)를 해석 가능하게 모델링한 시도"
</p>
